# @lofty87/types

common, model types defined only for lofty87's project

## 1. Installation

```bash
npm install --save @lofty87/types
```

## 2. Usage

#### 1. common types

```ts
import { AsyncFunction, Function, Object } from '@lofty87/types/common';
```

#### 2. model types

```ts
import { ModelPartial } from '@lofty87/types/model';
```
